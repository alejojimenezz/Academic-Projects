# Automatización de ejecución de programas

## Acceso a propiedades

1. Click derecho sobre el ícono del programa.
2. Seleccionar "Propiedades".

Para el documento se selecciona el programa "Visual Studio Code".

## Visual Studio Code

- **Destino:** `"C:\Users\ThinkPad\AppData\Local\Programs\Microsoft VS Code\Code.exe"`. La ruta anterior permite ejecutar el programa desde DOS.
- **Teclado de método abreviado:** `Ctrl + Mayus + V`
- **Archivo .bat:** Ejecuta el comando de *Destino* para abrir VSCode desde DOS.

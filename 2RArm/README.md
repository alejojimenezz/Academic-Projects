# Proyecto Servomecanismos

Mecanismo de brazo 2R, usando microcontrolador ESP32 para controlar los motores con encoder, y MatLab para generar simulaciones y gráficas.

Código del ESP32 realizado en Arduino IDE

Carpeta [TrayectoriaTrebol](./TrayectoriaTrebol/) incluye las gráficas en MatLab.

Demás capetas incluyen archivos de códigos de Arduino IDE para el ESP32.
